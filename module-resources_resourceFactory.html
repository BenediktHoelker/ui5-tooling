(inner) createWorkspace(parameters) → {DuplexCollection}

Source:
  • fs/lib/resourceFactory.js, line 133
Creates a Workspace A workspace is a DuplexCollection which reads from the project sources. It is used during the build process to write modified files into a seperate writer, this is usually a Memory adapter. If a file already exists it is fetched from the memory to work on it in further build steps.
Parameters:
Name Type Description
parameters Object
Properties
Name Type Attributes Default Description
reader AbstractReader The reader
name string <optional>
 "vir & fs source" Name of the collection
virBasePath string <optional>
 "/" Virtual base path
Returns:
DuplexCollection which wraps the provided resource locators
Type
DuplexCollection
